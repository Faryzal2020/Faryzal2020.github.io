Tank Ordnance Penetration Test Tool

A tool to tests WW2 tank munitions against a customizable RHA type armor plate. The data are gathered from War Thunder and the test results are not 100% accurate. Backend of the tool is entirely written only with Javascript. The data interpolation of the penetration values and ricochet chances are done using polynomial interpolation with Everpolate library.
